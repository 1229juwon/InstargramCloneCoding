## Features
1. 로그인, 회원가입, 로그아웃, 비밀번호 잊어버림 (이메일 확인 포함)
2. 페이스북과 인스타그램처럼 빠른 로그인 (앱이 로그인을 기억함)
3. 팔로우, 언팔로우, 프로필 조회.
4. 카메라 또는 갤러리에서 사진 게시
  * 게시물 좋아요
    * 게시물의 모든 좋아요 보기
  * 게시물에 댓글 달기
    * 게시물의 모든 댓글 보기
5. 프로필 편집 (사용자 이름, 소개, 소셜 링크, 이름).
6. 친구와 대화.
7. 이모티콘, 텍스트로 메시지 보내기.
8. 알림 받기.
9. 사용자 검색
  * 사용자 이름 기반으로 검색
10. 카메라 또는 갤러리에서 스토리 게시
  * 조회수
    * 스토리의 모든 조회수 보기
  * 삭제
    * 스토리 삭제
11. 오프라인 기능
